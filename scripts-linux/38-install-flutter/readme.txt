38-install-flutter
==================
let's start now 2026-04-26 (Asia/Kuala_Lumpur)

Title:    Flutter SDK
Method:   1) snap install --classic flutter (preferred — auto-updates)
          2) Tarball fallback: download stable release from storage.googleapis.com
                              into ~/.local/share/flutter, symlink into ~/.local/bin
Deps:     curl, git, unzip, xz-utils, zip, libglu1-mesa (apt)
Verify:   flutter --version

Note: First 'flutter doctor' run is left to the user (interactive Android SDK choices).
